To resolve merge confilicts:

1. git fetch origin
2. try with git merge origin/main if this does not work
    try git merge origin main --allow-unrelated-histories

3. fix the individual files which have conflicts.
4. add the changes to the staging area with git add .
5. commit the changes using git commit -m "commit message"
6. git push -u origin main to push the changes to the remote.

=======================To rollback to a remote version of the current repo=========================
To rollback to a remote version of the current repo:

1. git fetch origin
2. git checkout origin/main
3. git reset --hard origin/main
4. git push origin main --force

This will fetch the latest version of the main branch from the remote repository and reset the local repository to that version. It will then push the changes to the remote repository, overwriting any changes that may have been made since the last fetch.
